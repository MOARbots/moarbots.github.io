MOARbots

Modular, Open-Source, Affordable Robots

Welcome to MOARbots, the Modular, Open-Source, Affordable Robots project.

This framework aims to provide a set of designs, code, ideas, and documentation to aid in the construction of sets of robots that are:

  • Modular and highly reconfigurable - all mechanical designs are parametrically defined by code and rendered in tools such as OpenSCAD.
  • Open-source - All components of the framework are released as open-source and all existing tools leveraged by the framework are open source.
  • Affordable - under $50 per robot, with some designs costing much less. That cost estimate includes ALL parts, including batteries. It does not include the external elements of the framework such as a webcam and server.

In addition, MOARbots:

  • Are designed to be made with 3d printed plastic, and/or with easy to source parts, such as nails, rubber bands, marbles, hot glue, and heat shrink.
  • Feature a wireless radio, so that robots can be used in many applications, including: Overhead camera based autonomous navigation
    • Distributed multi-agent systems
    • Remote control
    • Small enough to run multiple robots in a standard arena whose size is defined by the field of view of a typical USB webcam mounted on the ceiling of a standard room
